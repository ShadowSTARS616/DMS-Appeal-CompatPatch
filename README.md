# DMS-Appeal-CompatPatch
  让更多mod服装（动力甲）能够使用失能机关需要特定体型装备的武器。可能会有点不平衡（过强）
  如果你不安装下列其他mod，则仅添加原版斥候甲的支持。
  
  目前支持：
  
  [OA]Ratkin Faction: Oberonia aurea
  
  [1.6] Moyo - from the depth
  
  Kiiro Race
  
  Combat extended

  服装能拿多大的武器是根据服装的动力甲相关描述、科技水平、负重加成、造价决定，以及亿点点私心（绮罗怎么没有动力甲，怎么会事呢）
  
  没有其他的主要是因为目前我只带了这些mod在玩（

  
  各服装能拿起的最大等级：

  绮罗夜行衣（kiiro）kiiro stealth clothes
  
  绮罗影袭者战服（kiiro）kiiro shadow strike suit
  
  绮罗勇者甲胄（kiiro）kiiro valor armor
  
  1.1体型：
  
  突袭作战裙甲（OA）Assault Combat Skirt Armor
  
  深海潜水服（moyo）
  
  斥候甲（原版）
  
  
  1.4体型：
  
  全防护突袭装甲（OA）Full protective assault armor
  
  深海高级潜水服（moyo）
  
  深海轻型皇家卫队装甲（moyo）
  
  海军甲（原版，仅用作对比）
  
  
  1.7体型：
  
  重装突袭作战护甲有/无披风（OA）Heavy assault armor A/B
